### Вариант №7. Сервис групп.

#### Добавить группу.

Информация требуемая для создания группы

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| year_create | Год создания группы | Обязательно | Целое число | от 2000 | |
| number | Номер группы | Обязательно | Целое число | от 1 | |
| prefix | Префикс | Обязательно | Строка | | |
| code | Шифр направления подготовки | Обязательно | Строка | вида `NN.NN.NN` (цифры, разделённые точками) | |
| class_number | После какого класса | Обязательно | Литерал (9, 11) | 9 или 11 | |
| tutor_id | id куратора | Не обязательно | Целое число | больше 0 | NULL |

Комбинация параметров: `year_create`, `number`, `prefix` и `class_number` должна быть уникальной

#### Изменить группу по ID

Входные параметры

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| group_id | ID группы (передается в URL) | Обязательно | Целое число | больше 0 | |
| tutor_id | id куратора | Обязательно | Целое число | больше 0 | |

**Примечание:** ID группы передается в path параметре, tutor_id - в теле запроса.

Выходные данные

| Параметр | Пояснение | Тип | Источник |
|---|---|---|---|
| year_create | Год создания группы | Целое число | Из БД |
| number | Номер группы | Целое число | Из БД |
| prefix | Префикс | Строка | Из БД |
| code | Шифр направления подготовки | Строка | Из БД |
| class_number | После какого класса | Литерал (9, 11) | Из БД |
| tutor_id | id куратора | Целое число или NULL | Из БД |
| name | Наименование группы | Строка (пример `3-2П9`) | Вычисляется автоматически на основе year_create, number, prefix, class_number |
| count_student | Количество студентов | Целое число | Получается из сервиса студентов (количество записей в таблице Student для данной группы) |
| students | Список студентов | Список целых чисел | Получается из сервиса студентов (ID студентов из таблицы Student для данной группы) |

#### Удаление группы по ID

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| group_id | ID группы (передается в URL) | Обязательно | Целое число | больше 0 |

**Результат выполнения:** Вернёт `True`, если группа была логически удалена (поле `is_active` установлено в `False`), иначе вернёт `False` (группа не найдена или уже удалена)

|---|---|---|---|---|

Информация о группе для пользователя

| Параметр | Пояснение | Тип |

#### Получение списка групп (с фильтрацией)

Параметры запроса

| Параметр | Пояснение | Тип | Ограничение |
|---|---|---|---|
| course_enumeration | Номер курса (точное совпадение) | Целое число | 1, 2, 3 или 4 |
| course_minimum_value | Номер курса (минимальное значение) | Целое число | от 1 до 4 |
| course_maximum_value | Номер курса (максимальное значение) | Целое число | от 1 до 4 |
| count_student_enumeration | Количество студентов (точное совпадение) | Целое число | от 0 |
| count_student_minimum_value | Количество студентов (минимальное значение) | Целое число | от 0 |
| count_student_maximum_value | Количество студентов (максимальное значение) | Целое число | от 0 |
| prefix | Префикс | Строка | |
| code | Шифр направления подготовки | Строка | вида `NN.NN.NN` |
| class_number | После какого класса | Литерал (9, 11) | 9 или 11 |
| tutor_id | id куратора | Целое число | `NULL` или 0 означает поиск групп без куратора, положительное число - поиск групп с конкретным куратором |

**Примечание:** Результат содержит только активные группы (`is_active = True`)

Выходные данные (список групп)

| Параметр | Пояснение | Тип |
|---|---|---|
| id | ID группы | Целое число |
| year_create | Год создания группы | Целое число |

**Примечание:** Для получения полной информации о группе используйте endpoint `GET /groups/{group_id}`

### ER-диаграмма

![ER-диаграмма](erd.png)

**Описание сущностей:**
- **Group** - группа студентов (год создания, номер, префикс, шифр, класс, куратор, активность, количество студентов)
- **Student** - связь студентов с группой (ID студента из внешнего сервиса, ID группы)

**Связи:** Group (1) ---> Student (M) - в одной группе может учиться много студентов
