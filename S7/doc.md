### Вариант №7. Сервис групп.

#### Добавить группу.

Информация требуемая для создания группы

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| year_create | Год создания группы | Обязательно | Целое число | ≥ 2000 | |
| number | Номер группы | Обязательно | Целое число | ≥ 1 | |
| prefix | Префикс (буквенное обозначение) | Обязательно | Строка | | |
| code | Шифр направления подготовки | Обязательно | Строка | строка из 8 символов в формате XX.XX.XX, где X - цифра | |
| class_number | После какого класса создана группа | Обязательно | Литерал (9, 11) | 9 или 11 | |
| tutor_id | ID куратора | Не обязательно | Целое число | > 0 | NULL |

**Уникальные комбинации параметров:** Комбинация `year_create`, `number`, `prefix` и `class_number` должна быть уникальной.

#### Изменить группу по ID

**Входные параметры (path parameter):**

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| group_id | ID группы | Обязательно | Целое число | > 0 |

**Входные параметры (body):**

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| year_create | Год создания группы | Не обязательно | Целое число | ≥ 2000 |
| number | Номер группы | Не обязательно | Целое число | ≥ 1 |
| prefix | Префикс | Не обязательно | Строка | |
| code | Шифр направления подготовки | Не обязательно | Строка | формат XX.XX.XX |
| class_number | После какого класса | Не обязательно | Литерал (9, 11) | 9 или 11 |
| tutor_id | ID куратора | Не обязательно | Целое число или NULL | > 0 либо NULL |

**Примечания:**
- `group_id` передается в URL, остальные параметры - в теле запроса
- Можно изменять одно или несколько полей одновременно
- Для удаления куратора передать `tutor_id: null`
- При изменении полей проверяется уникальность комбинации `year_create`, `number`, `prefix`, `class_number`

**Выходные данные:**

| Параметр | Пояснение | Тип |
|---|---|---|
| year_create | Год создания группы | Целое число |
| number | Номер группы | Целое число |
| prefix | Префикс | Строка |
| code | Шифр направления подготовки | Строка |
| class_number | После какого класса | Литерал (9, 11) |
| tutor_id | ID куратора | Целое число или NULL |
| name | Наименование группы (вычисляется автоматически из полей year_create, number, prefix, class_number) | Строка (пример `3-2П9`) |
| count_student | Количество студентов (динамически получается из сервиса студентов) | Целое число |
| students | Список ID студентов (динамически получается из сервиса студентов) | Список целых чисел |

#### Удаление группы по ID

**Входные параметры:**

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| group_id | ID группы | Обязательно | Целое число | > 0 |

**Результат выполнения:** Вернёт `True`, если группа была закрыта (логически удалена через установку флага `is_active = False`), иначе вернёт `False`.

#### Получение списка групп (с фильтрацией)

**Параметры запроса (фильтрация):**

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| course_enumeration | Курс обучения (точное совпадение) | Не обязательно | Целое число | 1, 2, 3 или 4 |
| course_minimum_value | Курс обучения (минимальное значение) | Не обязательно | Целое число | ≥ 1 |
| course_maximum_value | Курс обучения (максимальное значение) | Не обязательно | Целое число | ≤ 4 |
| count_student_enumeration | Количество студентов (точное совпадение) | Не обязательно | Целое число | ≥ 0 |
| count_student_minimum_value | Количество студентов (минимальное значение) | Не обязательно | Целое число | ≥ 0 |
| count_student_maximum_value | Количество студентов (максимальное значение) | Не обязательно | Целое число | ≥ 0 |
| prefix | Префикс группы | Не обязательно | Строка | |
| code | Шифр направления подготовки | Не обязательно | Строка | формат XX.XX.XX |
| class_number | После какого класса | Не обязательно | Литерал (9, 11) | 9 или 11 |
| tutor_id | ID куратора | Не обязательно | Целое число или NULL | NULL - без куратора, >0 - конкретный |

**Алгоритм расчета курса:**
- Курс = Текущий учебный год - Год создания группы + 1
- Учебный год начинается с 1 сентября
- Если текущая дата < 1 сентября, текущий учебный год = текущий год - 1
- Текущая дата и учебный год определяются сервером на момент выполнения запроса

**Выходные данные (сокращенный ответ):**

| Параметр | Пояснение | Тип |
|---|---|---|
| id | ID группы (первичный ключ) | Целое число |
| year_create | Год создания группы | Целое число |

**Примечания:**
- Возвращаются только активные группы (is_active = True)
- Для получения полной информации (name, count_student, students) используйте `GET /groups/{group_id}`
- Сокращенный формат выбран для оптимизации производительности при работе с большим списком

### ER-диаграмма

![ER-диаграмма](erd.png)

**Сущности:**
- **Group** (хранимые поля): id (PK), year_create, number, prefix, code, class_number, tutor_id, is_active
- **Group** (вычисляемые): name, count_student, students
- **Student**: id_student (PK), id_group (FK → Group.id)

**Связи:** Group (1) ---> Student (M) - один студент может учиться только в одной группе
