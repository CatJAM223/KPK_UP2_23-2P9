### Вариант №7. Сервис групп.

#### Добавить группу.

Информация требуемая для создания группы

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| year_create | Год создания группы | Обязательно | Целое число | от 2000 | |
| number | Номер группы | Обязательно | Целое число | от 1 | |
| prefix | Префикс | Обязательно | Строка | | |
| code | Шифр направления подготовки | Обязательно | Строка | вида `NN.NN.NN` (цифры, разделённые точками) | |
| class_number | После какого класса | Обязательно | Литерал (9, 11) | 9 или 11 | |
| tutor_id | id куратора | Не обязательно | Целое число | больше 0 | `NULL` |

Комбинация, параметров: `year_create`, `number`, `prefix` и `class_number`, должна быть уникальной

#### Изменить группу по ID

Входные параметры

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| tutor_id | id куратора | Обязательно | Целое число | больше 0 | |

Выходные данные

| Параметр | Пояснение | Тип |
|---|---|---|
| year_create | Год создания группы | Целое число |
| number | Номер группы | Целое число |
| prefix | Префикс | Строка |
| code | Шифр направления подготовки | Строка |
| class_number | После какого класса | Литерал (9, 11) |
| tutor_id | id куратора | Целое число или `NULL` |
| name | Наименование группы | Строка (пример `3-2П9`) |
| count_student | Количество студентов | Целое число |
| students | Список студентов | Список целых чисел |

#### Удаление группы по ID

Вернет `True` если группа была закрыта, иначе `False`

#### Информация о группах

Параметры запроса

| Параметр | Пояснение | Тип | Описание |
|---|---|---|---|
| course_enumeration | Номер курса (точное число) | Целое число | ровно "20" |
| course_minimum_value | Номер курса (перечисление "от") | Целое число | от "20" |
| course_maximum_value | Номер курса (перечисление "до") | Целое число | до "20" |
| count_student_enumeration | Количество студентов (точное число) | Целое число | ровно "20" |
| count_student_minimum_value | Количество студентов (перечисление "от") | Целое число | от "20" |
| count_student_maximum_value | Количество студентов (перечисление "до") | Целое число | до "20" |
| prefix | Префикс | Строка | |
| code | Шифр направления подготовки | Строка | вида `NN.NN.NN` (цифры, разделённые точками) |
| class_number | После какого класса | Литерал (9, 11) | 9 или 11 |
| tutor_id | id куратора | Целое число | 0 - если куратор не назначен |

Результат содержит только активные группы

Информация о группах для пользователя

| Параметр | Пояснение | Тип |
|---|---|---|
| id | id группы | Целое число |
| year_create | Год создания группы | Целое число |
