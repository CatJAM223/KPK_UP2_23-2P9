### Вариант №7. Сервис групп.

#### Добавить группу.

Информация требуемая для создания группы

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| year_create | Год создания группы | Обязательно | Целое число | от 2000 | |
| number | Номер группы | Обязательно | Целое число | от 1 | |
| prefix | Префикс | Обязательно | Строка | | |
| code | Шифр направления подготовки | Обязательно | Строка | вида `NN.NN.NN` (цифры, разделённые точками) | |
| class_number | После какого класса | Обязательно | Литерал (9, 11) | 9 или 11 | |
| tutor_id | id куратора | Не обязательно | Целое число | больше 0 | NULL |

Комбинация параметров: `year_create`, `number`, `prefix` и `class_number` должна быть уникальной

#### Изменить группу по ID

Входные параметры

| Параметр | Пояснение | Обязательность | Тип | Ограничение | Значение по умолчанию |
|---|---|---|---|---|---|
| group_id | ID группы (передается в URL) | Обязательно | Целое число | больше 0 | - |
| tutor_id | id куратора | Обязательно | Целое число | больше 0 | - |

**Примечания:**
- ID группы передается в path параметре, `tutor_id` - в теле запроса
- При изменении группы обязательно указать новый `tutor_id` (даже если нужно удалить куратора - передать NULL)
- Поле `tutor_id` не имеет значения по умолчанию при изменении

Выходные данные

| Параметр | Пояснение | Тип | Источник данных |
|---|---|---|---|
| year_create | Год создания группы | Целое число | Из БД |
| number | Номер группы | Целое число | Из БД |
| prefix | Префикс | Строка | Из БД |
| code | Шифр направления подготовки | Строка | Из БД |
| class_number | После какого класса | Литерал (9, 11) | Из БД |
| tutor_id | id куратора | Целое число или NULL | Из БД |
| name | Наименование группы | Строка (пример `3-2П9`) | **Вычисляется автоматически** на основе `year_create`, `number`, `prefix`, `class_number`. Не может быть задано извне |
| count_student | Количество студентов | Целое число | **Получается из сервиса студентов** (количество записей в таблице Student для данной группы). Обновляется автоматически |
| students | Список студентов | Список целых чисел | **Получается из сервиса студентов** (ID студентов из таблицы Student для данной группы). Обновляется автоматически |

#### Удаление группы по ID

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| group_id | ID группы (передается в URL) | Обязательно | Целое число | больше 0 |

**Результат выполнения:** Вернёт `True`, если группа была закрыта (логически удалена, поле `is_active` установлено в `False`), иначе вернёт `False` (группа не найдена или уже закрыта)

#### Информация о группе по ID

| Параметр | Пояснение | Обязательность | Тип | Ограничение |
|---|---|---|---|---|
| group_id | ID группы (передается в URL) | Обязательно | Целое число | больше 0 |

Выходные данные (полная информация о группе)

| Параметр | Пояснение | Тип |
|---|---|---|
| year_create | Год создания группы | Целое число |
| number | Номер группы | Целое число |
| prefix | Префикс | Строка |
| code | Шифр направления подготовки | Строка |
| class_number | После какого класса | Литерал (9, 11) |
| tutor_id | id куратора | Целое число или NULL |
| name | Наименование группы | Строка (пример `3-2П9`) |
| count_student | Количество студентов | Целое число |
| students | Список студентов | Список целых чисел |

#### Получение списка групп (с фильтрацией)

**Параметры запроса (фильтрация)**

| Параметр | Пояснение | Тип | Ограничение |
|---|---|---|---|
| course_enumeration | Номер курса (точное совпадение) | Целое число | 1, 2, 3 или 4 |
| course_minimum_value | Номер курса (минимальное значение) | Целое число | от 1 до 4 |
| course_maximum_value | Номер курса (максимальное значение) | Целое число | от 1 до 4 |
| count_student_enumeration | Количество студентов (точное совпадение) | Целое число | от 0 и выше |
| count_student_minimum_value | Количество студентов (минимальное значение) | Целое число | от 0 и выше |
| count_student_maximum_value | Количество студентов (максимальное значение) | Целое число | от 0 и выше |
| prefix | Префикс группы | Строка | - |
| code | Шифр направления подготовки | Строка | вида `NN.NN.NN` |
| class_number | После какого класса создана группа | Литерал (9, 11) | 9 или 11 |
| tutor_id | ID куратора | Целое число или NULL | `NULL` - группы без куратора, число >0 - группы с конкретным куратором |

**Важно:** 
- Номер курса вычисляется на основе `year_create` и текущей даты (курс = текущий учебный год - год создания + 1)
- Результат содержит **только активные группы** (где `is_active = True`)
- Для поиска групп без куратора необходимо передать `tutor_id = NULL`

**Выходные данные (список групп)**

| Параметр | Пояснение | Тип |
|---|---|---|
| id | ID группы | Целое число |
| year_create | Год создания группы | Целое число |

**Примечание:** В списке возвращаются только идентификатор и год создания группы для экономии трафика. Для получения полной информации о конкретной группе (включая `name`, `count_student`, `students`) используйте endpoint `GET /groups/{group_id}`

### ER-диаграмма

![ER-диаграмма](erd.png)

**Описание сущностей:**
- **Group** - группа студентов (поля: year_create, number, prefix, code, class_number, tutor_id, is_active, count_student)
- **Student** - связь студентов с группой (поля: id_student - ID из внешнего сервиса, id_group - внешний ключ к Group)

**Связи:** Group (1) ---> Student (M) - в одной группе может учиться много студентов
